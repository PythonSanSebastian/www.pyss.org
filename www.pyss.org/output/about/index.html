About

Asociacion de Ciencias de la Programacion Python San Sebastian (ACPySS)

Python San Sebastian Society (ACPYSS) is a Basque (Spanish) non-profit organization which organizes Python San Sebastian conferences and supports the Python community in the Basque area.

The ACPYSS was formed in 2014, with the mission to extend the use of Python and to build a community of Python users in the Basque area.

Organization details

The ACPySS Society board can be reached by email via info@pyss.org . This is an open email to all the board members.
